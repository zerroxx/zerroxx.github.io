Internship Game Designer and Developer at Wild Sea.

As game designer/developer I had to create a set of children games surrounding a theme.
Ease of use and playability were my key concerns while desinging games. The games had to be fun, but also challenging enough to actually challenge the players.
All games for the project where made in Javascript/Phaser.

Prototyping procedural AI.

Instead of creating full fledge games, I like to test small ideas, see if it works, so I can reuse it later on.
This project was slightly inspired by an old tv-show called 'Oban Star Racers', I like the flowiness of the vehicles, and wanted to recreate this with an AI that could work on random and procedural generated roads.

About me

Met een sterke achtergrond in softwareontwikkeling en een artistieke inslag, combineer ik technische bekwaamheid met een creatieve benadering. Ik geloof dat de beste software niet alleen functioneel is, maar ook esthetisch aantrekkelijk en intuïtief te gebruiken moet zijn. Mijn doel is om boeiende digitale ervaringen te creëren die mensen inspireren en een positieve impact hebben.

Ik heb ervaring met het werken aan diverse projecten, variërend van webapplicaties tot games en alles daartussenin. Ik geniet ervan om uitdagingen aan te gaan en oplossingen te bedenken die zowel technisch geavanceerd als gebruiksvriendelijk zijn.
